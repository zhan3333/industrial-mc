# minecraft 工业服务器端

minecraft 版本: 1.12.2

包含存档信息、配置信息，服务器使用 Sponge 水桶服，参考地址: https://www.mcbbs.net/thread-786074-1-1.html

## 如何运行服务端

1. 安装 Java jre8
2. 编辑 run.sh ，修改 java 可执行文件所在路径
3. 运行 ./run.sh
